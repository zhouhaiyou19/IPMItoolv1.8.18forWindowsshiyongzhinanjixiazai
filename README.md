## 使用方法

### 基础命令示例

- **查看帮助**: `ipmitool help`
- **读取传感器数据**: `ipmitool sensor`
- **查询FRU信息**: `ipmitool fru`
- **远程开关机**: 使用如下格式，具体命令依据实际情况调整
  - 开机: `ipmitool -I lanplus -H [IP地址] -U [用户名] -P [密码] power on`
    - 关机: `ipmitool -I lanplus -H [IP地址] -U [用户名] -P [密码] power off`

    请替换方括号内的内容为你自己的设备信息。
